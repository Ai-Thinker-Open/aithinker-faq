ESP8266 如此多的编程方法，有SDK/Arduino/Lua/MicoPython等，我到底选择哪个?
-----------------------------------------------

  即使更多的编程方法，其调用的都是C为底层的，但是根据项目需求来选型编程方式，能使你的项目快速落地即可。我司推荐使用SDK编程开发，


  - 采用C语言编程，跨平台移植性强；
  - 如果使用 cmake 需要将例程中 CMakeLists.txt 文件内的链接配置一同复制。
