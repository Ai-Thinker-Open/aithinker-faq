Ai-WB2系列模组支持哪些开发方式
---------------------------
AT开发，window环境和liunx环境开发

Ai-WB2-32S模组与ESP32-S模组有什么区别，软硬件能否兼容
--------------------------------------------
Ai-WB2-32模组与ESP32-S模组封装兼容，但是引脚定义不一样；软件上所使用的芯片也不一样，WB2-32S使用的是BL602芯片，ESP32模组使用的是ESP32D0WD芯片

WB2模组支持WAPI协议吗
--------------
支持，不过需要通过二次开发实现

WB2系列模组在做干扰规避测试项，表格上因填写什么参数
--------------------------
选择监测与避让

请问Ai-WB2-12F模组重新烧写固件的时候，EN引脚的电平设置情况如何？
---------------------------
当软件提示 “Please Press Reset Key” 的时候，将EN引脚拉低之后，再拉高

请问有没有支持入网阿里云的WiFi、蓝牙双模模组？
-----------------------------------
Ai-WB2系列模组均支持的，可通过AT指令入网阿里云 `(操作过程见链接)  <https://blog.csdn.net/Boantong_/article/details/112971695?ops_request_misc=%257B%2522request%255Fid%2522%253A%2522167141430716800215099422%2522%252C%2522scm%2522%253A%252220140713.130102334.pc%255Fblog.%2522%257D&request_id=167141430716800215099422&biz_id=0&utm_medium=distribute.pc_search_result.none-task-blog-2~blog~first_rank_ecpm_v1~rank_v31_ecpm-1-112971695-null-null.nonecase&utm_term=%E9%98%BF%E9%87%8C&spm=1018.2226.3001.4450>`__
，指令与文档中描述的指令是一样的；

请问一下，我这边Ai-WB2-12F模组是否支持lua语言开发？
------------------------
Ai-WB2-xx系列模组均不支持lua语言开发，仅8266系列支持；

请问贵公司是否有rj45以太网网络透传功能的模组？
------------------------
没有。

请问程序存储空间与我申请的flash空间是否会有冲突？
----------------------------
不会，空间的分化由分区表决定，编译器中已做了防冲突功能；

WB2系列模组可以用ESP-IDF开发吗？
-----------------------
不可以，芯片方案不同

WB2-07S模组天线是蓝牙的还是WIFI的，还是两个共用天线
-------------------------
BLE和WIFI共用一条天线，分时复用

请问WB2-32S是否支持arduino开发？
------------------------
不支持

请问WB2模组是用的什么内核？
-----------------------
RISC-V

请问WB2模组支持空中升级么？
---------------------
支持OTA

请问WB2模组有16M闪存的么？
----------------------
通用的是2M与4M版本，16M需求的需联系商务定制；

WB2系列模块能传输水下的视频吗
-----------------
不支持，水下会吸收一部分的WIFI信号
