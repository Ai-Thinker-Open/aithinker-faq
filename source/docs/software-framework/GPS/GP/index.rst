--------------

GP-02出货配的天线是无源天线还是有源天线
--------------
无源天线

GP-02开发板上的圆形器件是什么？实际电路设计能否不接？型号是什么？
--------------
圆形器件是法拉电容，不是纽扣电池，型号为XH414H。实际电路设计可以不接

GP-01和GP-02有什么区别
----------------------
只是封装不同，其他软件功能都一样

GPS模组上的VCC_RF引脚是做什么用途使用
-------------
给有源天线引脚进行供电

GP-02和BG-02有什么区别
--------------------
芯片厂商不同

GP-01的串口有两个吗
------------------
没有两个，只有一个串口

GPS上的VRTC引脚安装3V纽扣电池有没有起作用
--------------------
VBAT有供电会有热启动功能，如果VBAT有供电，GPS定位后断电，再次上电会有RTC时间输出。

GPS获取的时间能精确到ms的吗？，目前模块是1s返回一次时间，时间格式：hhmmss.sss，.sss这个显示是0，怎么设置才能看到值？
----------------
GPS都是按整秒输出的，不会有ms输出。

如果GPS模块由于外界地理环境的原因，暂时收不到数据，你们的GPS模块是否仍然可以送GPS本地的时间给MCU
--------------
GPS从定位到不定位时，模块仍然有时间输出，如果模块没有定位过是没有时间的。

GP-02模组能否使用GK9501指令集
-------------------------
不可以，两者芯片方案不同，不能兼容使用

GPS模块无法定位该如何排查
---------------------
首先检查是否在室外测试，室内肯定无法定位的，其次检查下供电，一般使用3.3V

GPS模块是主动发送定位信息的吗
---------------------
是的

请问GP-01模组串口通信协议是哪一种协议？
----------------------
NMEA协议，下载该资源（），其中包含协议文档；
 .. image:: gp.jpg

GP-02能外接GPRS模块或者4G模块传递消息吗
-------------------------
可以外接，通过串口的方式

GP-02模块可以导引无人机飞行吗
------------------
模块不支持导引功能

GPS模组可以获取网络时间吗
-----------------
不可以，不过可以通过搜星的方式获取UTC时间，然后自行转换

GPS模组只能使用有源天线吗
---------------
有源天线和无源天线都可以使用
