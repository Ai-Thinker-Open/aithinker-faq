PB-02开发工具能用arduino么
------------------------------------------------------------------------
  不能用arduino，它开发是使用keil的

PB-02开发板可以在arduino IDE开发吗？
------------------------------------------------------------------------
  不行哦，这个是使用keil5进行开发的

PB-02模块usb供电吗
------------------------------------------------------------------------
  USB既可以供电也可以当串口使用

pb02这个模组可以用jtag下载吗
------------------------------------------------------------------------
  不支持，它是通过串口下载的

PB-01开发板做照明等中心主机用？
------------------------------------------------------------------------
  可以的

PB-02模组供电后是不是就可以工作了？就可以通过蓝牙设备搜索到模组了吗？
------------------------------------------------------------------------
  供电后就能搜到蓝牙信号，https://blog.csdn.net/Boantong_/article/details/109474628

TB系列与PB系列在软件上能否兼容使用？
-----------------------------------------------
程序不兼容，但是BLE测试软件可以使用同一个

TB系列模组中SIGMESH协议跟私有MESH协议有什么区别？
----------------------------------------
SIGMESH协议的特点就是协议统一，所有采用SIGMESH协议的设备都采用了其中一套协议。而私有MESH则是协议独立。


PB-03F模组一包数据最大支持大多？
----------------------------------------
TB/PB系列模组一包数据的大小由MTU参数决定，最大支持254B。

PB01/02模组连接天猫精灵在设置三元组有哪些注意的地方
----------------------------------------
设备ID需要通过在线转换工具将十进制转换成十六进制并在前面补00凑够8个字节长度
