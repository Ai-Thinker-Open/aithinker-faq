PB-02开发工具能用arduino么
------------------------------------------------------------------------
不能用arduino，它开发是使用keil的

PB-02开发板可以在arduino IDE开发吗？
------------------------------------------------------------------------
不行哦，这个是使用keil5进行开发的

PB-02模块usb供电吗
------------------------------------------------------------------------
USB既可以供电也可以当串口使用

PB-02这个模组可以用jtag下载吗
------------------------------------------------------------------------
不支持，它是通过串口下载的

PB-01开发板做照明等中心主机用？
------------------------------------------------------------------------
可以的

PB-02模组供电后是不是就可以工作了？就可以通过蓝牙设备搜索到模组了吗？
------------------------------------------------------------------------
供电后就能搜到蓝牙信号，https://blog.csdn.net/Boantong_/article/details/109474628

TB系列与PB系列在软件上能否兼容使用？
-----------------------------------------------
程序不兼容，但是BLE测试软件可以使用同一个

TB系列模组中SIGMESH协议跟私有MESH协议有什么区别？
----------------------------------------
SIGMESH协议的特点就是协议统一，所有采用SIGMESH协议的设备都采用了其中一套协议。而私有MESH则是协议独立。


PB-03F模组一包数据最大支持大多？
----------------------------------------
TB/PB系列模组一包数据的大小由MTU参数决定，最大支持254B。

PB01/02模组连接天猫精灵在设置三元组有哪些注意的地方
----------------------------------------
设备ID需要通过在线转换工具将十进制转换成十六进制并在前面补00凑够8个字节长度

蓝牙调试助手可以给TB-02开发板发送AT指令控制IO口的电平吗
---------------------------------
出厂AT固件不支持，需要对AT固件做二次开发实现

PB-03F模组支持BLE5.0么？
--------------
支持，PB-03系列模组支持BLE5.2，该协议向下兼容；

如何实现手机控制TB-02的LED开关？
--------------
调用GPIO相关指令即可操作LED等的开关状态。

PB-02能传音乐吗
--------------
不能，该蓝牙没有音频芯片

PB03出厂默认AT固件最长能设置蓝牙广播名称为多少个字节
----------
12个字节

PB-01、02如何对接天猫精灵
-------------------------
烧写支持对接天猫精灵固件并通过指令设置天猫精灵平台提供的三元组

PB01如何外接带IPEX座子的天线
--------------
默认天线采用半孔焊盘方式，可以在主板上设计一个IPEX端口方便进行外接


PB系列的SDK代码用什么开发平台
----------------------
keil5

PB-03F模块的信号强度与通信距离能到多少
--------------------------------
信号强度10dbm，通讯距离空旷环境100米左右

PB-03F自带32.768K晶振吗
-------------------
内部芯片自带

TB-02的复位引脚是高电平复位还是低电平
--------------------------------
低电平复位

TB/PB系列模组能否支持SIG_MESH_1.0.1版本
--------------------------------
可以支持。

PB-03M模组能否贴片焊接
--------------------------------
可以贴片焊接。


BLE模块哪款支持自组网
--------------------------------
TB系列模组，需要跑私有MESH协议

PB02模组出厂AT固件支持对接天猫精灵吗
----------------
不支持，需要通过串口重新烧写天猫精灵固件

PB-03模组有内部分32.768k晶振么？
----------------------------
有的

PB-03可以实现蓝牙透传么？
---------------------
可以

PB-01模组能否在没有网关的情况下实现mesh自组网？
----------------------------------------
不可以，节点的数据需要网关上传

TB-02模组的MASH组网SDK开源么
---------------------------
开源的，SDK仓库链接为：https://github.com/Ai-Thinker-Open/Telink_SIG_Mesh

TB-03F V8固件，V9固件和AT MESH固件有啥区别
--------------------
V8 V9固件为AT基本功能固件，不具备MESH功能；AT MESH固件支持组网通信

PB-03能否能实现透传么
-------------------
该模组具有透传功能，详情参考说明文档。该模组支持二次开发，详情访问我司官方DOCS系统。

请问下PB-03可以二次开发么，用的哪一个开发环境呢？
----------------------------------------------
支持二次开发，用的开发环境是keil MDK。

TB-04模组能否与其他蓝牙设备进行通讯？
------------------------------------
通过二次开发可以实现；

TB系列模块天猫和小米的音响都支持吗
--------------------------
目前仅支持天猫精灵音响

请问一下PB-03F的RX0、TX0串口电平是TTL还是232？
------------------
是TTL电平

TB/PB系列模块可以跟电脑通信吗
-----------------
可以，不过需要购买专门的蓝牙适配器进行调试

PB-02模组上的RST引脚复位是高电平还是低电平
-----------------------------
低电平复位

PB-03F模组在烧录固件过程中掉电了，有办法恢复么？
---------------------------------
重新刷固件即可

TB系列模组支持OTA升级吗
-------------
支持，可以通过APP的方式进行升级

请问一下，PB-03F模组二次开发，GPIO中断能用吗？
------------------
可以的

你好，请问一下PB-03F模组在刷机的时候掉电了，还能回复吗？
----------------
重新刷固件即可；

PB系列能否与WB2系列进行蓝牙相互通讯
-----------------------
可以相互通讯

PB-03F可以接出IIC接口吗
------------------
可以

PB-03可以使用vscode软件开发吗
---------------------
可以，在vscode上安装keil插件

PB-03F开发板可以测蓝牙的指标吗 
-----------------------
不可以，测试蓝牙指标建议使用专门的测试设备

PB03模块组网的时候，手机可以加入MESH组网中吗
------------------------
手机只能当配网者

请问TB-02蓝牙模块不适用APP或者小程序的情况下能否连接手机自带的蓝牙？
-----------------------
不能

请问PB-02的MESH组网是用的阿里的SDK么？
--------------------
不是阿里的，SDK获取仓库链接：
`参考链接  <https://github.com/Ai-Thinker-Open/AiThinker-Open-PB-BleMesh>`__

请问TB-02模组的天猫固件是否上电即可连接天猫？
------------------------------
需要实现配置三元组，具体使用方法见说明文档：
`参考链接  <https://blog.csdn.net/Boantong_/article/details/104649680>`__


请问蓝牙模块可修改名称么？
-----------------------
使用修改名称的指令修改即可；注：修改以后配置从机


请问贵公司有支持BT/BLE双模模组么？
----------------------
没有

PB系列模块可以当普通蓝牙进行配对吗
------------------------
可以，不过仅支持配对，不支持通信，通信需要下载蓝牙调试助手

TB-04可以配棒状天线吗
---------------------
可以，不过需要用户自行焊接IPEX座子

PB-03F模块如何同时烧录AT固件和自己写的程序
-------------------------
在AT固件的工程中进行二次开发

有支持功率调节WIFI/蓝牙模块吗
---------------
ESP系列和TB PB系列都支持调试功率

PB01模块广播指令是同时吗还是依此发送
------------------
广播指令是同时发送的

请问PB-02模组是否有苹果端的蓝牙调试助手软件？
---------------------
在苹果商店上下载nrfconnect APP。

请问：PB-02模组对接ESP32-G网关的固件要用哪一个？
-----------------------------
需烧录特定固件，请联系安信可工作人员获取固件。

PB-03模组AT模式和透传模式如何切换
----------------------
透传切换到AT是发送+++字符串   AT切换到透传就发送AT+TRANSENTER 进入蓝牙透传模式

TB-03F蓝牙通信一个模块做主模块可以连接多少个从模块
------------------------
默认出厂AT固件仅支持一个  如果需要链接多个得需要烧写MESH固件

PB-03模块用mesh网络的话，支持ESP32-G网关吗
---------------------
支持使用

PB-03F模块支持蓝牙小程序扫码连接控制吗
---------------------
模组支持，需要对模块进行片上开发实现
