GPS定位获取到的数据如何转换？（针对A9G、EC-01G以及GP-01/GP-02模组）
--------------
答：获取到GNGGA经纬度数据需转换公式：data/100 + data(整数部分)%100/60 + data(小数部分)/600000。
获取到的数据为WGS84坐标系，还需转换为GCJ02坐标系。

NB-Iot系列模组支持短信发送吗？
--------------
EC系列模组支持短信发送，前提是物联网卡开通短信功能。

安信可的NB-Iot模组支持PSM模式吗？
--------------
支持。前提：搭载的物联网卡支持该模式。

EC-01G模组定位精度是多少
--------------
10米

EC-01G定位信息获取和NB-IOT的交互是共用一个串口还是独立分开的串口
--------------
共用同一个串口

EC-01与EC-01G的区别
--------------
EC-01与EC-01G除了GPS功能外，其他没有任何区别

EC-01G模组适配arduino吗
--------------
不适配

NB模块能否在国外使用
--------------
可以，只要附近有支持NB的基站就能使用

NB模组是否支持连接私有MQTT服务器
------------------------------
支持连接，但需使用移到物联网卡

NB模组中AT+SKTCONNECT指令是否支持域名连接
-------------------------------------
不支持，仅支持IP地址


NB模组上的pin24引脚是什么功能
------------------------
1.8V输出

哪款串口模块出厂固件默认发命令的波特率是9600的
------------------------
EC系列模块、GPS系列模块、RA-08系列模块

NB模块出现ERROR:308应该从哪方面排查问题
------------------------
首先检查是否有锁卡，其次在排查下天线问题
