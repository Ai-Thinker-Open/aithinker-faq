GPS定位获取到的数据如何转换？（针对A9G、EC-01G以及GP-01/GP-02模组）
--------------
答：获取到GNGGA经纬度数据需转换公式：data/100 + data(整数部分)%100/60 + data(小数部分)/600000。
获取到的数据为WGS84坐标系，还需转换为GCJ02坐标系。

NB-Iot系列模组支持短信发送吗？
--------------
EC系列模组支持短信发送，前提是物联网卡开通短信功能。

安信可的NB-Iot模组支持PSM模式吗？
--------------
支持。前提：搭载的物联网卡支持该模式。

EC-01G模组定位精度是多少
--------------
10米

EC-01G定位信息获取和NB-IOT的交互是共用一个串口还是独立分开的串口
--------------
共用同一个串口

EC-01与EC-01G的区别
--------------
EC-01与EC-01G除了GPS功能外，其他没有任何区别

EC-01G模组适配arduino吗
--------------
不适配

NB模块能否在国外使用
--------------
可以，只要附近有支持NB的基站就能使用

NB模组是否支持连接私有MQTT服务器
------------------------------
支持连接，但需使用移到物联网卡

NB模组中AT+SKTCONNECT指令是否支持域名连接
-------------------------------------
不支持，仅支持IP地址


NB模组上的pin24引脚是什么功能
------------------------
1.8V输出

哪款串口模块出厂固件默认发命令的波特率是9600的
------------------------
EC系列模块、GPS系列模块、RA-08系列模块

NB模块出现ERROR:308应该从哪方面排查问题
------------------------
首先检查是否有锁卡，其次在排查下天线问题

EC-01G模组是否可以将定位信息上传到电脑实时查看？
------------------------------------------
实现方式有很多，我们提供两个例子：一个是入网阿里云实时上传定位信息；第二种方式是实时上传定位信息至安信可公众号；

EC-01G模组支持北斗定位么？
------------------------
支持的

EC-01G模组的定位信息刷新时间为多少？
------------------------------
刷新时间为一秒两次

EC-01F开发板烧录失败，软件提示：preamble no response，是什么原因？
----------------------------------------------------------
证明通讯过程有问题，直接用安卓数据线连接开发板的micro USB口进行烧录即可

NB系列模组支持国外使用吗
------------------------
支持，但是分地区，北美/欧洲的可以，日本的不行

EC-01F开发板原理图上的D1,D4,D5,D6分别是什么器件
---------------------------------------------
这几个都是ESD器件，起到防静电的作用

我买的EC-01F模组，个别地方信号很好，个别地方信号很差是什么原因？
--------------------------------------
EC-01F模组是通过NB信号进行通讯的，信号的好与差与当地基站NB信号布局有关。

EC-01F模组要如何进入PSM低功耗模式？
-----------------------------
需搭配支持PSM功能的sim卡，在配置进入PSM即可。支持该功能的卡为：Cmnbiot6

EC系列的模组与MCU进行数据传输是接串口0还是串口1
------------------------------------
接串口1

EC-01G与以前A9脚位是完全兼容的吗
-----------------------------
脚位间距兼容，引脚定义不一样

GPS模块可以同时支持GPS+北斗吗
-----------------------
可以同时支持

NB开发板有没有USB-OTG功能
------------------
这个没有，板子上USB口只是当串口使用

NB模组如果查询CSQ值为99该怎么办
--------------------------
需要先入网成功再去查询

NB模组支持agps和星历加速吗
---------------------
不支持

你好，请问一下EC-01F发送指令“AT+CSQ”返回99错误码是什么原因？
-------------------------
若是发送“AT+CGATT?”返回为0，且“AT+CGATT=1”没有返回信息的话，表示卡没有流量了，连不上网络。

请问一下EC-01F模组的天线与地之间短接蜂鸣器会响正常吗？
----------------
正常的；

NB模块中的B3 B5 B8频段具体是多少
--------------------
B3上行频段：1710MHz-1785MHz 下行：1805MHz-1880MHz；B5上行频段：824MHz-849MHz 下行：869MHz-894MHz；
B8上行频段：880MHz-915MHz 下行：925MHz-960MHz

NB模块可以发短信吗
-------------------
不支持发短信

EC-01G进入深度睡眠后，IO引脚需要拉高多长时间才能唤醒
-------------------
10ms

EC-01G开发板有ADC功能吗
----------------
没有

EC-01模组能否上电自动入网并自动连接TCP连接保存，在掉线后能否自动连接
-------------------------
能够自动入网，但是不能自动连接TCP

NB模块掉线了会有提示吗
--------------
有的

请问，EC-01G能否实现GPS、北斗定位并将定位信息发送到指定IP服务上？
------------------------
答：可以实现

请问，EC-01F模组是否有基站定位功能？
--------------------
没有

请问一下NB模块能否在香港使用？
-----------------------
NB模块仅支持大陆的移动与电信。

请问WB2-32S与ESP32-S的固件也是兼容的吗？
----------------------
二者串口指令功能以及串口电路是兼容的，芯片不同，固件也不同，不能互相烧录对方的固件；

请问EC-01G模组的AT指令与普通电话卡是一样的吗？
-----------------------
NB模块的使用需配特定的NB功能SIM卡；

请问NB-IoT与2G/3G/4G/5G移动网络之间的区别是什么？
-------------------------
NB-IoT是窄宽带的应用，用于传输小数据量的应用；而2G/3G/4G/5G移动网络用于大数据量的传输，如文件传输，视频下载等等；

EC-01开发板可以使用HTTP协议吗
--------------------
可以使用

EC-01F模组支持打电话发短信吗
---------------------
模组不支持电话和短信功能

EC-01G支持定位吗
-------------------
支持，通过GPS芯片搜星定位

请问NB模组可以用手机卡么？
----------------------------
只要支持NB协议即可

请问为何EC-01G开发板接好天线以后不能获取到GPS定位信息？
-----------------------
要室外搜星。

请问NB系列模组是否可以实现信息回传功能？
--------------------------
可以通过MQTT等协议功能实现信息回传。

NB系列开发板是不是只需要接USB线即可？
----------------------------
只需要接MicroUSB数据线即可做功能驱动。

NB模块支持eSIM卡吗
----------------------
模块不支持eSIM

EC-01G模块可以实现北斗对时功能吗
------------------
通过GPS功能进行实现

EC-01G开发板可以接电池吗
-----------------
可以接，但是需要自行设计电池供电电路

EC-01G模组支持双模定位的AT指令是哪个
----------------
本身GPS指令支持双模定位

NB模组支持MQTT5.0版本通信协议吗
----------------
不支持5.0版本，目前仅支持3.1.1版本

EC系列模组能打电话和发短信吗
------------
不能，因为NB使用的是专用的物联网卡，没有打电话和发短信的服务

请问NB-IOT模组的通讯范围多广
-------------------
15KM；

请问NB-IoT模组有盲区么？
---------------
有的，使用场景距离基站太远则无法完成数据交互；

请问EC-01F模组支持5G么？
-------------------
支持5G；

请问EC-01G模组在sleep 1的时候唤醒需要操作wakeup引脚么？
---------------------
sleep1位睡眠监听机制，无需操作wakeup引脚的方式唤醒，任何接收均可唤醒模组，包括无线接收；

NB模块带不带基站定位功能？
--------------------
不带，NB模块目前只有GPS定位

EC-01G开发板在室内能不能接收到卫星信号
-----------------
室内不能接收，因为这个是使用GPS芯片进行定位，必须到室外进行接收卫星信号
