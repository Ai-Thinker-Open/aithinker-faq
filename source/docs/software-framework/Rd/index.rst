Rd-01模组二次开发资料中，哪些是蓝牙那边的资料？
----------------------
SDK中的Ai-Thinker-WB2子模块中application下Bluetooth即为蓝牙相关用例。

Rd-01模组能接入天猫精灵平台么？
----------------
不能。

请问：Rd-01模组可以用arduino做开发吗？
--------------
不支持。

Rd-01雷达模组可以监测出空间中人的数量吗，最多可以监测到几个人
----------------------
可以监测人的数量，需做二次开发实现，最多能监测到多少个人得看代码处理逻辑

Rd-01模块探测角度可调吗
-------------------------
模块不可以调整探测角度

Rd-01模块上面有人体感应芯片吗
---------------------
有，这款模组是由WIFI+BLE+雷达芯片组成的，可以进行人体感应

Rd-01雷达模组不接WIFI天线能否使用
---------------------------
模块上雷达芯片的天线是板载的，可以使用雷达功能

Rd-01模块可以感应多远距离，安装在空调内合适吗
------------------------------------
感应距离5米，可安装在空调内进行使用

Rd-01模块能检测到静止的人吗
--------------------------
模块可实现高灵敏度的人体存在状态感应，识别运动和静止状态下的人体

请问Rd-01可以按照WB2的AT指令实现组网么?
----------------------
不可以，雷达驱动与AT固件是两套东西

请问雷达的最短检测距离是？
----------------------------
0.75米；

Rd-01模组可以刷esphome固件吗
--------------
不可以刷，内部芯片方案不同，Rd-01模组使用的是BL602系列芯片，并非ESP系列芯片

雷达系列模组能区分物体和人体吗
---------------------
不能区分，仅支持监测是否有人无人

Rd-01雷达模组可以监测人体心跳和呼吸吗
----------------------
不可以监测心跳和呼吸，模组是可感知区域内是否有运动或者微动的人体

雷达24g和10g在性能上有什么差异吗
---------------------
10g只能监测运动的人且没有距离信息，24g可以监测运动和静止的人体且有对应的距离信息
