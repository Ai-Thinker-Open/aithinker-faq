语料命令词最多支持多少条？
----------------------
150条


唤醒词最多支持多少条？
------------------
10条


VC系列模组电路设计时候，需不需要外加功放芯片？
--------------------------------------------
不需要


VC系列模组支持以下哪些烧录方式？
------------------------------------------------
JTAG调试器和串口烧录


VC系列模组的电压电流要求是？
-------------------------------------------------
供电电压 3.6V-5V，供电电流>500mA 


中英版本唤醒/命令固件支持自学习吗？
-------------------------------------
支持


支持粤语/河南话等方言吗？
---------------------------------
不支持


VC系列和之前的VB系列语音模组，开发环境/下载器是一样的吗？
---------------------------------------------------
VC使用开发平台是Linux ,VB是Windows CDK IDE，各自有自己的下载器下载


VC系列模组接麦克风推荐购买哪种
----------------------------
可以按照灵敏度-27db的标准

VC系列模组咪头的拾音最远能达到多少米
--------------------------------
最远5米。
