--------------

语料命令词最多支持多少条？
----------------------
150条


唤醒词最多支持多少条？
------------------
10条


VC系列模组电路设计时候，需不需要外加功放芯片？
--------------------------------------------
不需要


VC系列模组支持以下哪些烧录方式？
------------------------------------------------
JTAG调试器和串口烧录


VC系列模组的电压电流要求是？
-------------------------------------------------
供电电压 3.6V-5V，供电电流>500mA 


中英版本唤醒/命令固件支持自学习吗？
-------------------------------------
支持


支持粤语/河南话等方言吗？
---------------------------------
不支持


VC系列和之前的VB系列语音模组，开发环境/下载器是一样的吗？
---------------------------------------------------
VC使用开发平台是Linux ,VB是Windows CDK IDE，各自有自己的下载器下载


VC系列模组接麦克风推荐购买哪种
----------------------------
可以按照灵敏度-27db的标准

VC系列模组咪头的拾音最远能达到多少米
--------------------------------
最远5米。

VC系列模组支持播报吗
---------------------
支持，不过没有TTS功能

VC系列模组使用语音开放平台是不能进入I2C开发吗？
------------------
不能开发I2C功能

VC系列模组支持自定义免唤醒命令词吗
------------------------
支持

VC系列开发板中文固件和英文固件是不是硬件通用的
-----------------------------
共用同一个硬件

VC模组在线创建产品SDK中有B2B3B6B7引脚，但是实际产品引脚定义没有这几个引脚，是怎么回事
--------------------------
B0 B1 B8 是特殊功能脚，B8是系统log 默认是输出脚波特率57600，B0 B1是JATG 烧录脚

VC系列所使用的咪头参数是多少
------------------
咪头的接收灵敏度为-31dB

VC-02模块支持永久唤醒吗
--------------
支持，需要通过开发实现，出厂固件没有实现该功能

请问VC语音开发板的串口是哪一种芯片？
----------------------
CH340
